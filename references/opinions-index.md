# Arizona Ethics Opinions — Topical Index

This index covers the **State Bar of Arizona Formal Ethics Opinions, 1985–2016**, issued by the State Bar's former Committee on the Rules of Professional Conduct (the "Ethics Committee"). The State Bar **no longer issues** formal ethics opinions; that function now belongs to the **Arizona Supreme Court Ethics Advisory Committee** (post-2016 opinions; see `opinions-index-eac.md` / azcourts.gov). Unauthorized-practice-of-law (UPL) advisory opinions are a separate State Bar series (see `upl-opinions.md`).

## How to read / retrieve full text

- Opinions are numbered `YY-NN` (e.g., `97-09` = the 9th opinion of 1997). The two-digit prefix encodes the year (85–99 → 1985–1999; 00–16 → 2000–2016).
- The **Src** column is the State Bar website's internal opinion id. Full text renders (JavaScript) at:
  `https://www.azbar.org/for-legal-professionals/ethics/ethics-opinions/?V=Opinions&OpinionId=<Src>`
- Full opinion text for harvested opinions is in `opinions-full.md`. If an opinion is in this index but not yet in `opinions-full.md`, retrieve it from the URL above or another trusted legal research source. **Never quote an opinion you have not actually read in full.**

## Reliability / status notes (READ FIRST)

- **Opinions are advisory and non-binding.** They do not bind the Arizona Supreme Court, the Presiding Disciplinary Judge, or any tribunal.
- **Rules drift.** Every opinion applies the Ethical Rules (and statutes/case law) **in effect on its publication date.** The ER numbering and text have changed over time (Arizona adopted the current ABA-style ER framework effective Dec. 1, 2003, with later amendments). An opinion citing pre-2003 rule numbers must be mapped to current ERs before quoting as current law. Confirm the current ER text in `rpc.md`.
- **Withdrawn / vacated / superseded** opinions are flagged in the Topic column where the State Bar so marked them (e.g., "WITHDRAWN OPINION," "Vacated," "Overrules," "Invalidated by," "Partially Withdrawn by"). Do not rely on a withdrawn opinion as current guidance.
- Topic tags are the State Bar's own opinion captions. They are descriptive, not a substitute for reading the opinion.
